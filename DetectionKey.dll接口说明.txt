检测密钥返回错误代码.

函数:Function Checking(szKey As String, ByRef ResultList As List(Of String)) As Integer
szKey:输入密钥
ResultList:返回检测结果，包括(密钥类型,密钥ID,密钥类型,错误代码,剩余次数,检测时间)


vb.net下使用举例,新线程下:

        Dim Thread As New System.Threading.Thread(Sub() func.Checking(szKey, ItemList))
        Thread.IsBackground = True
        Thread.Start()
        Do Until ItemList.Count > 0
            Application.DoEvents()
        Loop
        Thread.Abort()
        Debug.Print(String.Join("*", ItemList))
        
 或者
 
     Private Function InlineAssignHelper(Of T)(ByRef Target As T, ByVal Value As T) As T
        Target = Value
        Return Value
    End Function
    
    
        Dim nFlag As Integer = 0
        Dim Thread As New System.Threading.Thread(Sub() InlineAssignHelper(nFlag, func.Checking(txtKey.Text, ItemList)))
        Thread.IsBackground = True
        Thread.Start()
        Do Until nFlag <> 0
            Application.DoEvents()
        Loop
        Thread.Abort()
